# sso-use-cookie

单点登录使用cookie的版本，项目使用gradle。一共有4个子系统，sso-vip是VIP系统，sos-main是主页系统，sos-cart是购物车系统，sos-login是登陆系统。

主要完成了单点登录和单点退出的逻辑
